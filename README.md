# Medical Disease Predictor

Приложение для предсказания риска сердечного заболевания по медицинским признакам.

## Что делает
- Принимает данные пациента через FastAPI.
- Возвращает предсказание модели.
- Поддерживает интерактивную проверку через `/docs`.

## Как запустить
1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Запустить сервер:
```bash
uvicorn api.app:app --reload
```

3. Открыть в браузере:
```bash
http://127.0.0.1:8000/docs
```

## Файлы проекта
- `api/` — FastAPI-приложение.
- `models/` — сохранённые модели.
- `src/` — вспомогательные модули.
- `heart-disease.csv` — данные.
